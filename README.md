# Primary Grade Calculator

A simple, interactive command-line Python application designed to calculate a student's average score across four primary school subjects and automatically assign a letter grade.

## 🚀 Features
* **Multi-Subject Input**: Accepts scores for Mathematics, English, Science, and Social Studies (SST).
* **Automated Grading**: Calculates the mean score and applies a standard grading scale (A to E).
* **Error Handling**: Uses a `try-except` block to catch invalid inputs (like letters or symbols) and prevent the application from crashing.

## 📊 Grading Scale
* **80 - 100**: Grade A
* **70 - 79**: Grade B
* **60 - 69**: Grade C
* **50 - 59**: Grade D
* **Below 50**: Grade E

## 🛠️ Prerequisites
* Python 3.x installed on your computer or a mobile Python runner (like Python CodePad).

## 💻 How to Run

1. Open your terminal, command prompt, or Python CodePad app.
2. Run the script using the following command:
   ```bash
   python main.py
   ```
3. Enter the student's name and their numerical scores when prompted.
